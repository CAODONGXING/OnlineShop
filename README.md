# OnlineShop



***php based websites project***

    
## Dependencies

AppServ : Apache + PHP + MySQL

Apache Web Server Version 2.4.25.   

PHP Script Language Version 5.6.30 & 7.1.1. 

MySQL Database Version 5.7.17. 

phpMyAdmin Database Manager Version  4.6.6. 

Phpstorm 2017.
